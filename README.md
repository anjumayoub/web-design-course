# Web Design Mastery

An 8-week, hands-on course covering web + SaaS design. Built as a single self-contained HTML app.

## Live preview

**[Open the course](https://anjumayoub.github.io/web-design-course/)** - https://anjumayoub.github.io/web-design-course/

## What's inside

- 8 weeks / 42 sections: UX foundations, grids and layout, web typography, color systems, components and forms, tables and dashboards, SaaS patterns and marketing sites, responsive design, accessibility and a capstone project
- Interactive quizzes with scoring and a final exam
- Study timer with per-section daily history and summary
- Sticky notes per section: rich text, images, voice recordings, export
- Text highlighting, glossary, media viewer with zoom, light/dark theme
- Practice exercises built around real Mobbin references

Progress, notes and highlights save locally in your browser.
